# Landkit_Symfony

Symfony project, with the bootstrap landkit theme integrated.

## Installation

```bash
cd Desktop && git clone https://github.com/C-PONS-DEV/Landkit_Symfony
```

## Usage

```bash
composer install
```

```bash
npm install
```

```bash
npm run build
```

```bash
symfony server:start
```
